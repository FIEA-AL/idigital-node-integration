# IDigital Node Integration

[![License][license-badge]][license-url]

> IDigital OpenID Connect integration for Node.js

## Usage

- Install package

```bash
# Yarn
$ yarn add @idigital/node-integration

# NPM
$ npm install @idigital/node-integration
```

## Development

- Clone the repo

```bash
$ git clone https://github.com/FIEA-AL/idigital-node-integration.git
```

- Install dependencies

```bash
$ yarn install
```
